# GitHub Tutorial

## 项目简介

这是一份系统性的 Git 和 GitHub 学习教程，涵盖了从基础概念到实践操作的完整内容。

## 内容目录

### 📚 [01-基础概念.md](01-基础概念.md)
- Git 的核心概念
- 版本控制原理
- 工作区、暂存区、本地仓库的关系
- 分支管理基础

### 🛠️ [02-Git命令详解.md](02-Git命令详解.md)
- 常用 Git 命令详解
- `git init`, `git add`, `git commit` 等基础命令
- 分支操作命令
- 合并与冲突解决

### 🚀 [03-GitHub-CLI使用.md](03-GitHub-CLI使用.md)
- GitHub CLI (gh) 工具介绍
- 仓库管理
- Pull Request 操作
- Issues 管理
- 命令行高效工作流

### 💡 [04-Claude-Code最佳实践.md](04-Claude-Code最佳实践.md)
- Claude Code 使用技巧
- AI 辅助编程最佳实践
- 提高开发效率的方法

## 适用人群

- Git 和 GitHub 初学者
- 希望系统学习版本控制的开发者
- 想要提高 Git 使用效率的程序员
- 对 AI 辅助编程感兴趣的学习者

## 学习建议

1. **按顺序学习**：建议从 01 到 04 依次阅读
2. **动手实践**：每学习一个概念，立即在本地实践
3. **记录笔记**：在实践中总结自己的理解
4. **反复练习**：Git 命令需要多次练习才能熟练
